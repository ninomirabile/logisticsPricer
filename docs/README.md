## 🆕 New Features Documentation

### Admin Dashboard
The comprehensive admin management interface provides:

- **📊 Dashboard Overview**: Real-time metrics and system status
- **🚚 Shipping Routes Management**: Create, edit, and manage transport routes
- **💰 Tariff Management**: Create, edit, and manage shipping tariffs
- **💵 Pricing Management**: Configure pricing requests and calculations
- **🇺🇸 USA Duties Management**: Handle US import duties and taxes
- **📈 Analytics & Reporting**: Performance metrics and business insights

**Quick Access**: http://localhost:5173/admin

### Shipping Routes Management
Comprehensive transport route management system:

- **🗺️ Route Creation**: Define origin-destination pairs with transport types
- **⏱️ Transit Time Calculation**: Base time, customs delays, port congestion
- **💰 Cost Management**: Base costs, customs fees, port fees, additional costs
- **📋 Requirements Tracking**: Documents, special handling, certifications
- **🚫 Restrictions Management**: Route-specific restrictions and limitations
- **📊 Route Analytics**: Statistics and performance metrics

**Quick Access**: http://localhost:5173/admin/shipping-routes

### Pricing Management
Advanced pricing request and calculation system:

- **📝 Request Management**: Create and manage pricing requests
- **🧮 Calculation Engine**: Automatic price calculation with breakdown
- **📊 Response Tracking**: Store and manage calculation results
- **📈 Statistics Dashboard**: Analytics and performance metrics
- **🔍 Advanced Filtering**: Search by city, product, HS Code, status
- **📄 Historical Data**: Complete request history and tracking

**Quick Access**: http://localhost:5173/admin/pricing

### USA Duties Module
Specialized functionality for US import operations:

- **🏛️ Duty Calculation**: Automated US customs duty calculations
- **📋 Tariff Classification**: HS code management and classification
- **💰 Tax Management**: Sales tax, excise tax, and other US taxes
- **📊 Compliance Reporting**: Regulatory compliance and documentation
- **🔄 Real-time Updates**: Live duty rate updates and currency conversion

**Quick Access**: http://localhost:5173/admin/usa-duties
